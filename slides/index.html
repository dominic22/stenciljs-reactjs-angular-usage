Was sind Web-Komponenten ?

  • Standard für universelle Komponenten
  • Erstellung von benutzerdefinierten HTML Elemente
  • Verwendbar in allen JavaScript Frameworks und Plain JavaScript
  • Ermöglichung von Shadow DOM

Shadow DOM

  • Funktionalität ist gekapselt und von weiterem Code vollständig getrennt
  • Styles innerhalb haben keine Auswirkung auf Komponenten außerhalb
  • Styles außerhalb haben keine Auswirkung auf Komponenten innerhalb

Was ist Stencil ?

  • Compiler für Web-Komponenten
  • Kein neues Frontend Framework
  • Stencil Komponenten bauen auf Web Komponenten auf
  • Basiert auf TypeScript und verwendet JSX
  • Open Source
  • Entwickelt von Ionic

Warum Stencil ?

  • Code wird von Compiler optimiert
  • Unterstützt alle gängigen Browser
  • Einfache API zur Erstellung von Web-Komponenten
  • Virtual DOM, JSX, und asynchrones rendern
  • 100% kompatibel zum Web Komponenten Standard

Stencil API

  • Lifecycle Hooks: Lebenszyklus Events einer Komponente
  • Dekoratoren: Eigenschaften einer Komponente

Dekoratoren - @Component

@Component({
	tag: 'st-button',
	styleUrl: 'st-button.less'
})
export class StButton {
}
  • Deklariert die Web-Komponente <st-button>

Dekoratoren - @Prop

import { Prop } from '@stencil/core';

export class StButton {
	@Prop() color: string;
	@Prop() icon: string;
	@Prop() label: string;
}
  • Deklariert ein Property oder Attribut

Dekoratoren - @State

export class StToggleButton {
	@State() open: boolean;

	@Listen('click', { capture: true })
	handleClick() {
		this.open = !this.open;
	}

	render() {
		return ;
	}
}
  • Interner Status einer Komponente

Dekoratoren - @Method

import { Method } from '@stencil/core';

export class StPopUp {
	@Method()
	async open() {
		// show the popup
	}
	@Method()
	async close() {
		// close the popup
	}
}
  • Methode die man über Referenz zur Komponente aufrufen kann

Dekoratoren - @Event

import { Event, EventEmitter } from '@stencil/core';

...
export class TodoList {

	@Event() todoCompleted: EventEmitter;

	todoCompletedHandler(todo: Todo) {
		this.todoCompleted.emit(todo);
	}
}
  • DOM Event welches die Komponente feuert

Framework Integration

Framework Integration - React

import React from 'react';
import ReactDOM from 'react-dom';
import App from './App';
import { applyPolyfills, defineCustomElements } from 'stencil-library-itd/loader';

ReactDOM.render(<App />, document.getElementById('root'));
registerServiceWorker();

applyPolyfills().then(() => {
	defineCustomElements(window);
});
  • defineCustomElements ist "Main" Funktion
  • Lädt Web Komponenten Bibliothek
  • Einmalig während Bootstrapping aufrufen

Framework Integration - React + TypeScript

declare namespace JSX {
	interface IntrinsicElements {
		[tagName:string]: any
	}
}
  • Verwendung benutzerdefinierter JSX Elemente
  • Custom Type Definition Datei verwenden

Framework Integration - Angular

import { BrowserModule } from '@angular/platform-browser';
import { CUSTOM_ELEMENTS_SCHEMA, NgModule } from '@angular/core';
import { FormsModule } from '@angular/forms';

import { AppComponent } from './app.component';

@NgModule({
	declarations: [AppComponent],
	imports: [BrowserModule, FormsModule],
	bootstrap: [AppComponent],
	schemas: [CUSTOM_ELEMENTS_SCHEMA],
})
export class AppModule {}
  • CUSTOM_ELEMENTS_SCHEMA in jedem Modul angeben, das Web Komponenten verwendet

Framework Integration - Angular

import { enableProdMode } from '@angular/core';
import { platformBrowserDynamic } from '@angular/platform-browser-dynamic';
import { AppModule } from './app/app.module';
import { environment } from './environments/environment';
import { defineCustomElements } from 'stencil-library-itd/loader';

if (environment.production) {
	enableProdMode();
}

platformBrowserDynamic().bootstrapModule(AppModule)
	.catch(err => console.log(err));
defineCustomElements(window);
  • defineCustomElements einmalig während bootstrapping aufrufen
